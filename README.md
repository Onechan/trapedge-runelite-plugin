# TrapEdge Plugin Hub Submission README V1

TrapEdge is a RuneLite plugin for one narrow job:

> warn when an OSRS flip looks profitable on paper but is actually fragile, fake-spread, thin-flow, or weak-exit bait.

## What the plugin does
- shows trap-oriented triage for watched flips
- gives the next action: `enter now`, `wait`, `small size only`, or `avoid`
- shows compact supporting context instead of only raw spread numbers
- includes memory/replay context as read-only evidence

## What it does not do
- no automation
- no buying/selling actions
- no bot behavior
- no in-game action execution
- no writeback/postmortem submission from the client

## Current design posture
This Plugin Hub-ready version is intentionally read-only.

The goal is a boring, reviewer-safe utility panel — not a broad platform shell.

## Best-fit user
- active GE flippers
- players already using RuneLite and flipping tools
- users who care about fake spreads, weak exits, and fragile setups

## One-line summary
TrapEdge is a warning layer for OSRS flips, not another generic tracker.
